# lit-genre-atlas — decision log

## Session 2026-07-06 (headless build, branch claude/literary-workflow-feedback-cdlwlt)

- **Origin**: explicitly user-requested ("Build a skill that holds a database
  of genres and successful creations from each genre"). Governance verdict:
  CREATE-NEW, project-specific (`lit-*` family). Neighbors: bmad-market-research
  (business sizing, not craft contracts), lit-teach (pedagogy; consults this
  atlas for genre data — boundary recorded both sides), lit-author-style-research
  (authors, not genres), bmad-competitive-features (software).
- **Data placement**: the LIVING database is the skill's own
  references/genre-atlas.md (seeded verbatim from the R8 research compile —
  21 genre/subgenre sections, fixed field template, [K] markers on
  compiler-knowledge entries, as-of dates). Deliberately NOT duplicated into
  docs/research/ — one source of truth; R5–R7 research snapshots live in
  docs/research/literary-pack/. Updates supersede entries in place.
- **Comps method**: references/comps-method.md distills R7 — the 4-step
  trope-fingerprint join, the full Playing-With mode taxonomy (mode stored,
  not just presence; opposed modes = anti-comps), coverage caveats, and the
  binding cautions (no checklist writing; tropes-are-tools; CC BY-NC-SA —
  names/graph usable, scraped page text never shipped).
- **Which-game rule**: the position intent encodes R6's central finding —
  prestige-language and bestseller-language are distinct, partially opposed
  targets; the skill asks which game before advising. The same rule is folded
  into writer-mode behavior (this commit) and docs/literary-pack.md.
- **Contract-check posture**: violations are highest-severity QUERIES, never
  fixes — the author may be deliberately cross-genre; evidence framing
  ("readers treat violation as betrayal") accompanies every contract finding.
- **Fixtures**: ga-1 embeds a short DESCRIPTION of the "A day of ruin"
  manuscript (premise + mechanics; no prose quoted) per the user's fixture
  approval; ga-2's manuscript premise is the user's own from this session.
- **Validation mode**: degraded in-session (trigger proxy + 3 artifact evals,
  two with live file access to the atlas); hermetic re-run via
  bmad-eval-runner when available.
- **Validation results (2026-07-06)**: trigger proxy 10/10 (incl. the two key
  boundaries: "teach me to write a mystery" → lit-teach, market sizing →
  bmad-market-research). Artifact evals 3/3 PASS — ga-1 contract check
  (missing progression flagged as highest-severity QUERY "choice or gap?",
  serial-cadence norms cited from atlas with directional framing, DCC
  positioning bonus); ga-2 comps (4 steps in order, Laconic verification
  taught, mode-based anti-comps named, fate-accepted-vs-defied mode fork
  surfaced as the decisive question); ga-3 which-game (progression payoffs
  prioritized for declared KU game with evidence, dated data caveats, final
  call left to the author).
- **Installer pack**: added the `writer` pack to scripts/wf-install.py
  (writer-mode + the three lit-* skills) — wf install selftest was failing
  with "non-manifest skills no pack claims"; the pack claim is also the
  installer-level realization of "fold everything into the writer mode pack".
