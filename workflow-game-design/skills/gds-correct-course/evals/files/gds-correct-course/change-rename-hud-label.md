# Triggering Change — Rename HUD Label

**Discovered during:** UI polish review, sprint 4.

**Issue:** The HUD bar under the health bar is labeled "Energy", but the GDD, the design team, and every other reference call this resource "Stamina". The label is just inconsistent terminology — it confuses new playtesters who look for a "Stamina" bar.

**Desired change:** Rename the HUD bar label text from "Energy" to "Stamina". Pure label/string change. No mechanic change: same bar, same max, same costs, same regen, same winded behavior. Nothing about how the resource works changes — only the displayed word.
