# PRD — TaskFlow MVP

## Vision

Enable individual contributors to capture, organize, and complete daily tasks with a minimal, distraction-free interface.

## Functional Requirements

- **FR-1** A user can create a task with a title and optional due date.
- **FR-2** A user can mark a task as complete.
- **FR-3** A user can view all incomplete tasks sorted by due date.
- **FR-4** A user can delete a task.

## Non-Functional Requirements

- **NFR-1** The task list must load in under 1 second for up to 500 tasks.
- **NFR-2** All task data must be persisted locally (no cloud sync in MVP).

## MVP Scope

In scope: FR-1 through FR-4 and NFR-1 through NFR-2.
Out of scope: collaboration, reminders, recurring tasks, mobile app.
