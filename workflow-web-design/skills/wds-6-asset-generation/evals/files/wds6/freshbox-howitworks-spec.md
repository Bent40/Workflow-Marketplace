# FreshBox — "How It Works" page spec + design system (excerpt for Asset Generation)

## Design System (tokens)
Colors:
- Background: Cream (#FBF7EF)
- Primary/CTA: Leaf Green (#2E7D4F), gradient to #3F9E66
- Text: Charcoal (#23211E)
- Secondary text: Stone (#6B6760)
- Success: #28C54A
- Error: #C0392B

Typography:
- Font: Inter
- Headlines: Bold, tight letter spacing
- Body: Regular, 16px base

Component styles:
- Buttons: rounded 10px, green gradient for primary, subtle shadow
- Cards: cream surface, soft shadow

## Page spec: How It Works (mobile 390px)
Purpose: reassure a busy parent that the weekly cycle is predictable and that skip/pause is penalty-free, then send them to plan selection.
Brand feel: warm, trustworthy, chef-credible.

Layout zones:
1. Header: FreshBox logo (left), "Sign in" link (right)
2. Hero: headline + one-line subhead
3. 3-step weekly-cycle explainer (numbered cards)
4. Skip/Pause reassurance band (must be unmistakable, before any signup)
5. "What's in a box" example card
6. Primary CTA: "Choose a plan"

Key copy:
- Hero headline: "Dinner, handled — every week."
- Hero subhead: "Chef-planned boxes delivered weekly. Skip any week, no penalty."
- Step 1: "Pick your plan"
- Step 2: "We deliver fresh ingredients + recipes"
- Step 3: "Cook in ~30 minutes"
- Skip band: "Going away? Skip or pause any week with one tap. No fees, ever."
- Box example title: "This week's box"
- Primary CTA: "Choose a plan"

Audience: busy two-income parents, 30-45.
Market: single English-speaking market.
