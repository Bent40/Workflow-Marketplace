# FreshBox — Page specs (excerpt for Design System extraction)

Two page specs are provided so shared components and tokens can be extracted.

## Page A: How It Works (mobile 390px)
- Primary button "Choose a plan": leaf-green gradient (#2E7D4F -> #3F9E66), white label, rounded 10px corners, subtle shadow, full-width on mobile.
- Numbered step cards: cream surface (#FBF7EF), soft shadow, charcoal text (#23211E), 16px body in Inter.
- Body font: Inter, 16px base; headline: Inter Bold.

## Page B: Plan Selection (mobile 390px)
- Primary button "Subscribe weekly": leaf-green gradient (#2E7D4F -> #3F9E66), white label, rounded 10px corners, subtle shadow, full-width on mobile.
- Plan cards: cream surface (#FBF7EF), soft shadow, charcoal text (#23211E), 16px body in Inter; selected state has a 2px leaf-green border.
- Secondary "Compare plans" text link: leaf-green text, underline on hover.

## Notes
- Colors used across both: cream #FBF7EF, leaf green #2E7D4F (and #3F9E66 gradient stop), charcoal #23211E, stone secondary text #6B6760.
- Both pages use the same primary button treatment (the only difference is the label).
- Both pages use a card pattern (cream surface, soft shadow).
