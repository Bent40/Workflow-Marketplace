# UX Spec — PetClinic Scheduler

## Screens

- **Booking calendar** — month/week view of available slots; tap a slot to book.
  Supports the booking flow (FR-2) and reschedule (FR-3).
- **My appointments** — list of upcoming appointments with reschedule/cancel
  actions.
- **Staff dashboard** — day grid of all appointments; drag-to-move to reschedule
  (FR-5).

Note: the UX spec does not describe any confirmation-email / notification
experience or screen.
