# UX Spec — SecureNotes

## Screens

### Lock Screen
- Master password field
- Unlock button
- No error/lockout state shown (lockout behavior not designed)

### Note List Screen
- List of note titles
- Search bar at top
- Add note button (FAB)

### Note Detail Screen
- Title field
- Body textarea
- Save / Delete buttons

## Coverage
- FR-1: Note detail screen covers create/edit/delete.
- FR-2: Lock screen covers authentication.
- FR-3: Search bar on note list screen.
